Welcome

This website explores the evolution of HTML, its importance in modern web development, and the role it plays alongside CSS and JavaScript. HTML provides the basic structure of webpages by organizing content such as headings, paragraphs, images, links, lists, and forms.

Since its creation, HTML has continued to develop in response to changes in technology and the growing needs of internet users. Modern HTML allows developers to create webpages that are more organized, accessible, interactive, and compatible with different devices.

The information presented throughout this website is based on research completed for IT 3203 - Intro to Web Development. The goal of the project is to explain how HTML developed and why it remains one of the most important technologies used on the World Wide Web.

Key Terms

The following terms are important for understanding the topics discussed throughout this website:

  • HTML: The language used to structure content on webpages.
  • HTML5: The modern version of HTML used by web developers.
  • CSS: A language used to control the design and appearance of webpages.
  • Semantic Elements: HTML elements that clearly describe their purpose.
  • Accessibility: Designing websites so they can be used by people with different abilities.
  • Responsive Design: Creating webpages that adjust to different screen sizes.
  • Web Development: The process of creating and maintaining websites.
