Research Sources

The following sources were used to research the history of HTML, modern web development, HTML5, CSS, and related web technologies. These resources provide reliable information from organizations and websites commonly used by students and web developers.

Works Cited

  • Tim Berners-Lee Proposal
    The original proposal that introduced the idea of the World Wide Web.
    https://www.w3.org/History/1989/proposal.html

  • MDN Web Docs - HTML
    A comprehensive reference explaining HTML elements, best practices, and modern web development.
    https://developer.mozilla.org/en-US/docs/Web/HTML

  • W3Schools HTML Tutorial
    A beginner-friendly resource used to learn HTML syntax, examples, and common webpage elements.
    https://www.w3schools.com/html/

  • WHATWG HTML Living Standard
    The official living specification describing the current HTML standard.
    https://html.spec.whatwg.org/

  • W3C HTML Overview
    An overview of HTML provided by the World Wide Web Consortium, the organization responsible for many web standards.
    https://www.w3.org/html/
