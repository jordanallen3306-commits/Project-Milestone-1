What Is HTML?

HTML, or HyperText Markup Language, is the standard markup language used to create and organize webpage content. It uses elements and tags to identify different types of information, including headings, paragraphs, images, links, tables, lists, and forms.

HTML gives a webpage its basic structure. For example, a heading is placed inside a heading element, while a paragraph is placed inside a paragraph element. Web browsers read these elements and use them to determine how the content should be displayed.

What Is CSS?

CSS, or Cascading Style Sheets, controls the visual appearance of a webpage. Developers use CSS to change colors, fonts, spacing, borders, backgrounds, alignment, and page layouts.

Separating design from structure makes websites easier to manage. Instead of adding the same styles directly to every HTML element, a developer can place the styling rules in one CSS file and connect that file to multiple webpages.

How HTML and CSS Work Together

HTML and CSS have different responsibilities, but they work together. HTML describes what appears on a webpage, while CSS controls how that content looks. A webpage can function without CSS, but it would usually appear plain and unorganized.

For example, HTML may create a navigation menu with several links. CSS can then place those links in a row, change their colors, add spacing, and create a hover effect. This combination improves both the organization and appearance of the website.

The Role of JavaScript

JavaScript adds behavior and interactivity to a webpage. It can respond to button clicks, check form answers, update text, display messages, and change webpage content without requiring an entirely new page to load.

In this project, JavaScript will be used to grade the quiz. HTML will create the quiz questions, CSS will style the quiz, and JavaScript will calculate the score and display feedback to the user.
