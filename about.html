About Me

My name is Jordan Allen, and I am an Information Technology student at Kennesaw State University. I am currently learning about website structure, design, and programming through IT 3203 - Intro to Web Development.

This project has allowed me to practice writing HTML, creating shared CSS styles, organizing multiple webpages, and using navigation links to connect an entire website.

Purpose of the Project

This website presents research about the evolution and importance of HTML in modern web development. It explains the history of HTML, its purpose, important web development concepts, and how HTML works with CSS and JavaScript.

The project is designed to demonstrate both research and technical skills. Each page focuses on a different topic while maintaining a consistent layout, navigation menu, color scheme, and footer.

Technologies Used

The website was created using three major web technologies:

  • HTML was used to organize the content and create the structure of each webpage.
  • CSS was used to control colors, spacing, layout, tables, images, navigation, and other visual elements.
  • JavaScript will be used to grade the interactive quiz and display feedback to the user.

Project Learning Goals

The main goal of this project is to develop a better understanding of how different web technologies work together. Building the website also provides experience with file organization, code comments, responsive design, accessibility, and testing webpages in a browser.

The final website will include an interactive quiz that allows users to test their understanding of the topics presented on the other pages.
